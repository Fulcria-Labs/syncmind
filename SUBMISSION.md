# SyncMind - Hackathon Submission

## Tagline
An offline-first AI research assistant that thinks locally, syncs globally.

## What it does
SyncMind is a research note-taking app where AI analysis happens seamlessly in the background while your data stays local-first. Create notes anywhere - online or offline - and SyncMind automatically generates summaries, tags, discovers connections between your ideas, and builds an interactive knowledge graph. Everything syncs across devices via PowerSync the moment connectivity returns.

## The Problem
Researchers, students, and knowledge workers accumulate notes across tools that don't talk to each other. Existing AI-powered note apps require constant internet connectivity, lose your work when offline, and don't help you discover hidden connections between ideas. There's no tool that combines the reliability of local-first with the intelligence of AI.

## How PowerSync Makes This Possible

PowerSync is not just a nice-to-have - it's the architectural foundation that makes SyncMind's core value proposition work:

1. **Instant local writes**: Notes save to SQLite in the browser with zero latency. No loading spinners, no "saving..." states. This is critical for a note-taking flow where every millisecond of friction breaks concentration.

2. **Sync Streams for granular data flow**: Three dedicated streams (`user_notes`, `note_connections`, `all_tags`) efficiently sync different data shapes. Notes sync bidirectionally, while AI-generated connections and tags flow from server to client after processing.

3. **The AI feedback loop**: User creates a note locally -> PowerSync syncs to PostgreSQL -> backend detects unprocessed notes -> Claude AI generates summary, tags, connections -> results written to PostgreSQL -> PowerSync syncs AI insights back to browser -> UI updates reactively via `useQuery`. The user never waits - they keep working while AI processes in the background.

4. **Offline-first AI degradation**: When offline, SyncMind falls back to local keyword extraction and extractive summarization (no API needed). When connectivity returns, full AI analysis replaces local approximations seamlessly via sync.

5. **Reactive queries everywhere**: Every component uses PowerSync's `useQuery` hooks. The knowledge graph, tag cloud, note list, and connection panels all update automatically when synced data arrives - no manual refresh needed.

## Key Features

- **Knowledge Graph**: Interactive force-directed visualization of note relationships. Drag nodes, click to navigate, hover for connection details. AI discovers relationships you didn't know existed.
- **Research Agent Chat**: Mastra-powered conversational agent with tools to search notes, find connections, and synthesize research across your entire knowledge base.
- **Smart Tag Cloud**: Visual topic frequency display. Click any tag to filter instantly. Tags generated by AI analysis of content.
- **Research Brief**: One-click AI synthesis across all your notes - get a high-level overview of your entire research.
- **URL Import**: Paste a URL and SyncMind fetches the content, creating a note automatically.
- **PWA/Installable**: Full Progressive Web App with service worker caching. Install on desktop or mobile for native-like offline experience.
- **Markdown Export**: Export any note with AI metadata as clean markdown.
- **One-Click Demo Data**: Judges can populate the app with 6 interconnected AI research notes instantly.

## Architecture

```
Browser (SQLite via WASM)           PowerSync Service
  |                                      |
  |-- Local reads (instant)              |-- Sync Streams (3 streams)
  |-- Local writes (queued)              |-- Conflict resolution
  |-- Reactive useQuery hooks            |-- Bidirectional sync
  |                                      |
  v                                      v
         PostgreSQL (source of truth)
                    ^
                    |
    Express Backend --> Mastra Agent --> Claude AI
      |                    |
      |-- JWT auth         |-- Note search tool
      |-- CRUD handler     |-- Connection finder tool
      |-- AI processor     |-- Research synthesizer tool
      |-- Seed endpoint
```

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Sync Engine | PowerSync (self-hosted, Sync Streams) |
| Frontend | React 18 + TypeScript + Vite |
| Local DB | SQLite (PowerSync WASM) |
| AI Agent | Mastra framework |
| AI Models | Claude (Anthropic) |
| Backend | Node.js + Express |
| Database | PostgreSQL + MongoDB (PowerSync storage) |
| Deployment | Docker Compose (full stack) |

## What makes this original

1. **AI + Sync feedback loop**: Most AI note apps are cloud-only. SyncMind is the first to use PowerSync's sync streams as the transport layer for an AI analysis pipeline, where insights flow back to the user's local database automatically.

2. **Graceful AI degradation**: Offline doesn't mean dumb. Local NLP (keyword extraction, extractive summarization) provides immediate value, then gets upgraded by cloud AI when sync resumes.

3. **Knowledge graph from AI connections**: Rather than manual linking (like Obsidian), SyncMind's AI automatically discovers semantic relationships and builds a navigable graph.

4. **Conversational research agent**: The Mastra-powered agent doesn't just answer questions - it uses tools to actively search your synced notes, find connections, and synthesize findings.

## How to run

```bash
git clone https://github.com/Fulcria-Labs/syncmind.git
cd syncmind
export ANTHROPIC_API_KEY=your_key
docker compose up -d
cd frontend && npm install && npm run dev
# Open http://localhost:5173 and click "Load Demo Data"
```
